# TaskFlow

TaskFlow is a full-stack task management application built to practice and demonstrate end-to-end web development skills. The project allows users to create, update, organize and track daily tasks through a clean and responsive interface.

The goal of this project was not just to build a task manager, but to work through the complete development workflow including frontend development, backend API design, database integration, version control, deployment, and production hosting.

## Features

* Create, edit and delete tasks
* Mark tasks as completed
* Search tasks instantly
* Filter tasks by status
* Due date management
* Progress tracking dashboard
* Responsive UI for desktop and mobile
* Persistent task storage
* REST API integration
